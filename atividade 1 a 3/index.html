Projeto Escolar

Problema escolhido

Esse projeto foi iniciado com um objetivo de ajudar a melhorar a organização da escola entre os alunos e os professores.

Calendário

O calendário vai ser utilizado para marcar

os eventos, datas de avaliações. E quando

clicar na data vai aparecer o horário, matéria,

quem vai ser o "organizador" e entre outros.

Avaliações

Nas avaliações é para os professores

organizarem quando vai acontecer a

avaliação, o horário e qual matéria vai ser.

Os alunos só podem visualizar.

Avisos

Os avisos é para ajudar na organização e

para aqueles que não receberam, não lembram

ou perderam os avisos. Então os avisos

serão enviados pelo apllicativo.

Funcionalidades do sistema

  • Cadastro dos alunos.
  • Cadastro de professores.
  • Calendário de avaliações e eventos.
  • Envio de avisos e comunicados.
  • Consulta de horários das aulas.
